
# 📚 Library Management System

## Overview
Library Management System is a simple console-based application developed in C using File Handling. It helps manage library operations such as adding books, searching books, issuing books, returning books, and maintaining records efficiently.

## Features
- Add new books
- Search books by ID or title
- Display all books
- Issue books
- Return books
- Modify book details
- Delete book records
- Store records permanently using file handling

## Technologies Used
- C Programming
- Structures
- Functions
- File Handling
- GCC / CodeBlocks
